The main Function in C++  
What is main?   
① The main function is the entry point of a program.  
② Every C++ program must contain exactly one main function — no more, no less.


```cpp
// The following 11 lines of code print "Hello World" on the screen

#include <iostream>
using namespace std;

int main() {

    cout << "hello C++" << endl;  // Display "hello C++" on the screen
    system("pause");              // Pause the program until a key is pressed

    return 0;
}

```


Program Output  
hello C++  
Press any key to continue . . .
